Лабораторная работа 1. Разработка пользовательского интерфейса (GUI) для языкового процессора







Цель работы -  Создание кроссплатформенного графического интерфейса (GUI) для языкового процессора в виде специализированного текстового редактора.















Сведения об авторе - Марченко А.Е. АП-326















Описание проекта: Текстовый редактор - простое приложение с графическим интерфейсом, предназначенное для создания и редактирования текстовых файлов. Предоставляется базовый набор функций для работы с текстом:
Создание, открытие, сохранение файлов. Возможность работать с текстом: редактирование, копирование, вырезать, выделение, отмена/повтор действия.














Используемые технологии: C# Windows Forms .NET Visual Studio.
