### 20) Построить РВ, описывающее целые числа и числа с плавающей точкой (разделитель точка).
(?<!\S)-?\d+(?:\.\d+)?(?!\S)

| Обозначение | Пояснение |
|-----------|---------------------|
| **(?<!\S)** | Перед числом нет непробельного символа (только начало строки, пробел, табуляция). |
| **-?** | Отрицательное или положительное число. |
| **\d** | Любая цифра от 0 до 9 |
| **+** | Означает, что цифра должна присутствовать хотя бы один раз |
| **(?:)** | Группирует выражение |
| **\.** | Десятичный разделитель (символ .) |
| **\d+** | Одна или более цифр после точки |
| **?** | Вся дробная часть (?:\.\d+) может отсутствовать (целое число) или присутствовать |

- Примеры строк, которые должны находиться;

| Текст | Найденные слова |
|-----------|---------------------|
| **123** | 123 |
| **1.23** | 1.23 |
| **1** | 1 |

- Примеры строк, которые не должны находиться;

| Текст | Найденные слова |
|-----------|---------------------|
| **a123** | Строка начинается с буквы |
| **ala** | Только буквы |
| **ala.123** | Строка начинается с букв |
| **123.ala** | После точки стоят буквы |

### 18) Построить РВ для поиска идентификатора, который может начинаться только с буквы a-zA-Z, знака доллара $ или подчеркивания _, оставшаяся часть символов идентификатора представляют собой только буквы a-zA-Z или цифры.
(?<!\S)[A-Za-z_$][A-Za-z0-9]*(?!\S)

| Обозначение | Пояснение |
|-----------|---------------------|
| **(?<!\S)** | Перед числом нет непробельного символа (только начало строки, пробел, табуляция) |
| **[A-Za-z_$]** | Первый символ идентификатора – только буква латиницы в любом регистре (A-Z или a-z), знак доллара ($) или подчёркивание (_) |
| **[A-Za-z0-9]*** | Оставшаяся часть – ноль или более символов, каждый из которых – буква латиницы (A-Za-z) или цифра (0-9) |
| **"*"** | Квантификатор ноль или более раз (оставшаяся часть может отсутствовать) |
| **(?!\S)** | Проверяет, что после идентификатора нет непробельного символа |


- Примеры строк, которые должны находиться;

| Текст | Найденные слова |
|-----------|---------------------|
| **x** | x |
| **$var** | $var |
| **_temp** | _temp |
| **abc123** | abc123 |

- Примеры строк, которые не должны находиться;

| Текст | Найденные слова |
|-----------|---------------------|
| **1abc** | Строка начинается с цифры|
| **!abc** | Строка начинается с недопустимого символа |
| **)Z** | Строка начинается с недопустимого символа |Вавав
| **Вавав** | Строка содержит недопустимые символы |

### 23) Построить РВ, описывающее время. Формат: ЧЧ:ММ:СС в 24-часовом формате с обязательным ведущим 0.
\b(0[0-9]|1[0-9]|2[0-3]):[0-5][0-9]:[0-5][0-9]\b

| Обозначение | Пояснение |
|-----------|---------------------|
| **\b** | Граница слова – позиция между символом «буква/цифра/подчёркивание» и символом «не буква/не цифра/не подчёркивание», либо начало/конец строки) |
| **(0[0-9] | 1[0-9] | 2[0-3])** | Группа, описывающая часы в 24-часовом формате с ведущим нулём: - 0[0-9] – от 00 до 09; - 1[0-9] – от 10 до 19; - 2[0-3] – от 20 до 23; Вертикальная черта "|" означает «или». |
| **:** | Двоеточие (разделитель между часами и минутами) |
| **[0-5][0-9]** | Минуты: первая цифра от 0 до 5, вторая – любая цифра (от 00 до 59) |
| **:** | Ещё одно двоеточие (разделитель между минутами и секундами) |
| **[0-5][0-9]** | Секунды – аналогично минутам (от 00 до 59) |
| **\b** | Граница слова в конце, чтобы время не было частью более длинной строки |

- Примеры строк, которые должны находиться;

| Текст | Найденные слова |
|-----------|---------------------|
| **00:00:00** | 00:00:00 |
| **09:15:30** | 09:15:30 |
| **23:59:59** | 23:59:59 |

- Примеры строк, которые не должны находиться;

| Текст | Найденные слова |
|-----------|---------------------|
| **24:00:00** | Часы больше 23 |
| **12:5:30** | Нет ведущего нуля у минут|
| **12:60:00** | Минуты больше 59 |
| **12:34:5** | Нет двух цифр у секунд |
